# LICHESS_GAMES_PROJECT
This project analyses chess game data using Tableau dashboards, Excel, and SQL. It explores patterns in win outcomes, player ratings, and game characteristics such as turn length. Interactive visualizations help reveal trends across rating categories and match results.
